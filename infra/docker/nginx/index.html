EV SaaS Platform API Gateway

Welcome to the EV SaaS Platform API Gateway. This gateway provides access to various microservices that power the EV SaaS Platform.

Available Services

Auth Service

Authentication and authorization service.

Base URL: /api/auth/

API Docs: API Documentation (JSON)

Documentation: Swagger UI | Direct Auth Swagger UI | Auth Service Swagger UI (Direct)

User Service

User management service.

Base URL: /api/users/

Station Service

EV charging station management service.

Base URL: /api/stations/

Billing Service

Billing and payment processing service.

Base URL: /api/billing/

Smart Charging Service

Intelligent charging optimization service.

Base URL: /api/smart-charging/

Notification Service

User notification service.

Base URL: /api/notifications/

Roaming Service

OCPI implementation for roaming.

Base URL: /api/roaming/

OCPI Endpoint: /ocpi/

API Documentation

Access the API documentation via:

  • Custom Swagger UI - Recommended for accessing all service APIs
  • Swagger UI - Standard Swagger UI from Auth Service
  • API Docs (JSON) - Raw API specification

Health Check

A simple health check is available at /health to verify the gateway is running properly.
